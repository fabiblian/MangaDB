# MangaDB

## Kurzbeschreibung
MangaDB ist eine Fullstack-App zur Verwaltung von Mangas. Die Anwendung enthält JWT-Authentifizierung, ein Rollenmodell mit `USER` und `ADMIN`, persönliche Lesestatus, ein Leaderboard sowie Reading Sessions.

## Tech-Stack
- Spring Boot
- MySQL / Docker
- React / Vite

## Projekt starten
- DB starten
- Backend starten
- Frontend starten
